Number Object

Number objects are used to work with numbers to do the arithmetic operations. Unlike many other programming languages, JavaScript does not define different types of numbers, like integers, short, long, floating-point etc. JavaScript numbers are always stored as double precision floating point numbers.

Creating a Number object

The Number() constructor is used to create a new number object. 
                
    var num = new Number(number);
Example: 
                
    var num = new Number(10);
    document.write(num);
    document.write(num*num);
    document.write(new Number('Hi'));
    document.write(new Number());
Output: 
        
    10
    100
    NaN
    0
When you give the non numeric value as a argument to the Number() constructor that will return NaN.
